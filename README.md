# Notebook — Algoritmos y Programación 1

Bienvenidos al notebook del curso Algoritmos y Programación 1. Este repositorio contiene materiales interactivos y ejemplos para las clases.  
Estado: trabajo en progreso — el contenido se actualizará y mejorará continuamente.

## Ver online (Binder)

Para ver el notebook online, se puede usar Binder.

[![Ejecutar en Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/untref-ayp1/notebook/HEAD)

Al hacer clic se lanzará una instancia en la nube con el contenido del repositorio; puede tardar unos minutos en arrancar.

## Cómo levantarlo localmente

Requisitos básicos:
- make instalado

Desde la raíz del repositorio, ejecutar:
```bash
make check start
```
- `make check` — ejecuta comprobaciones de herramientas
- `make start` — arranca el servidor de notebooks para trabajar en el entorno local.
